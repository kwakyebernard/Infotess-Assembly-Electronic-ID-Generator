# Infotess-Assembly-Electronic-ID-Generator
Generates unique electronic IDs for Course Representatives in Infotess efficiently. The system allows the administrator to input details such as ID number, full name, class and status ( main/depity ) and automatically produces a standardized digital ID card ready for printing or saving for digital use.
# The project's technologies used
HTML, CSS and JavaScript
jsPDF
GitHub pages
# Features
Dynamic form for inputting course representative's details
Clean UI with INFOTESS branding
Instant print and PDF export option or save for digital use
Responsive design for multiple screen sizes
